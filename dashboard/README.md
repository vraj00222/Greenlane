# GreenLane Dashboard

Web dashboard for tracking your sustainable shopping journey.

## Quick Start

```bash
pnpm dev
```

Opens on http://localhost:3002

## Features

- View eco-scores and scan history
- Track achievements and XP progress
- Compare sustainability metrics
- Leaderboard and community features

## Tech Stack

- Next.js 15
- Tailwind CSS
- shadcn/ui components
- SWR for data fetching
